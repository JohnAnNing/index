第一次提交
git init创建暂停区
git add 文件名————添加文件到暂停区
可能需要输入邮箱跟姓名
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
git commit
仓库
git status 查看状态
git commit -m "备注内容"
git log -oneline显示提交状态
git commit --amend -m "覆盖提交"
git checkout 清空工作区
git reset --hard 版本号
git reflog 查看完整记录
git add. 提交
git rm 文件名 ————————删除工作区 和仓库里的文件
git rm --chached 只删除工作区文件
git diff ：比较本地和暂存区的文件区别
